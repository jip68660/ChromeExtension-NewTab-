# ChromeExtension-NewTab-
This is a project called "JJamtab", which is chrome new tab extension. We desinged for soldiers in South Korea. This extension contains basic features of new tab, and military related features for soliders with providing useful information about their military life, calculate their discharge date, save the photo, and more. API and bootstrap are used for features and design.

Thank you. 

# Significant features
background & theme, 7 search engines, weather API, calculate discharge date in a various ways, calculate relationship date for girlfriend/boyfriend, 3 photo album, and more
